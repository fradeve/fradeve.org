Poster AGU 2010

Poster presented at AGU Fall Meeting in San Francisco.

This poster was made using LaTeX with the Beamerposter package. I’ve modified one of the default themes to my own needs (and taste). If you are interested in using (and modifying) this template you can download the tarball with the LaTeX source code and figures:

  • paoloAGU2010.tar.bz2
  • paoloAGU2010.pdf (81 MB)

To generate a PDF poster first decompress the tarball:

$ tar xjf paoloAGU2010.tar.bz2

and then compile the source code:

$ cd paoloAGU2010    
$ pdflatex paoloAGU2010.tex

That’s all!
